== Create a Message Template

A message template allows you to customize how the message is sent to your audience. You can add custom styles and text using FreeMarker.

A new template can be added by clicking on the icon:plus[type=fas, role=ui-button green] button.

image::messengers/message-templates-home.png[Message Templates Home, width=1200]

=== Add Message Template

image::messengers/message-templates-add.png[Message Templates Add, width=1200]

==== Form Fields

[.api]
[field]#Id# [optional]#Optional#::
A unique UUID is auto generated if left blank.

[field]#Name# [required]#Required#::
For display only. Name of the message template.

[field]#Type# [optional]#Optional#::
The type of message template. Options include: SMS, Kafka, and Generic.

=== Template

image::messengers/message-template-section.png[Message Templates, width=1200, role=top-cropped bottom-cropped]

==== Form Fields

[.api]
[field]#Default Text# [required]#required#::
This text is displayed in each message. The default text accepts replacement variables.
Valid replacement variables are listed below.

==== Replacement Variables

[.api]
[field]#application# [type]#[Application]# [since]#Available since 1.21.0#::
The Application object, see the Application API for field definitions.

[field]#code# [type]#[Number]#::
The unique code intended to be used by the Two Factor authenticator.

[field]#email# [type]#[String]#::
Email address associated with the `user`.

[field]#mobilePhone# [type]#[Number]#::
Mobile phone number associated with the `user`.

[field]#tenant# [type]#[Tenant]# [since]#Available since 1.18.2#::
The Tenant object, see the Tenant API for field definitions.

[field]#user# [type]#[User]#::
The User object, see the User API for field definitions of a User.

=== Localization

image::messengers/message-templates-add-localization.png[Message Templates Add Localization, width=1200, role=top-cropped bottom-cropped]

==== Form Fields

[.api]
[field]#Id# [optional]#Optional#::
You can enter localization options here. See our guide to localization link:/docs/v1/tech/themes/localization[here]

Click `Add localization` and fill additional details as outlined below.

=== Localization Details

image::messengers/message-templates-add-localization-popup.png[Message Templates Add Localizations Popup, width=600, role=box-shadow top-cropped bottom-cropped]

==== Form Fields

[.api]
[field]#Locale# [Required]#Required#::
Enter the language for which you are localizing the message.

[field]#Template# [optional]#Optional#::
The translated message.

You may check how your message will look by clicking `Preview`
