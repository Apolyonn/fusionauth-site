== Setup a Twilio Messenger

Here you can add a Twilio messenger to send SMS messages. +

image::messengers/add-messenger.png[Messenger Home Screen, width=1200]

To create a new generic messenger, navigate to `icon:comments[type=fas] Messengers`. +
Simply click `icon:plus[type=fas, green,  solid] Add Twilio Messenger`

=== Add Twilio Messenger

image::messengers/add-messenger-twilio.png[Add Messenger Twilio, width=1200]

Just complete the following fields:

==== Form Fields

[.api]
[field]#Id# [optional]#Optional#::
A unique UUID is auto generated if left blank.

[field]#Name# [required]#Required#::
For display only. Name of the Twilio messenger.

[field]#URL# [required]#Required#::
Provided by Twilio. This is the URL of the Twilio messaging service you wish to connect to.

[field]#Account SID# [required]#Required#::
Provided by Twilio.

[field]#Auth token# [required]#Required#::
Provided by Twilio.

[field]#From phone number# [optional]#Optional#::
The outgoing phone number of your messenger service.

[field]#Messaging service Id# [optional]#Optional#::
Provided by Twilio and if often used in conjunction with the Copilot service.

[field]#Debug enabled# [optional]#Optional#::
Will allow for more verbose debugging. Output is stored in the events log.

You also can test your Twilio config to make sure that you have it set up correctly.
