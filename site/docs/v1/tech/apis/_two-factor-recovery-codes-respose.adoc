==== Response Body

This response body is only returned when the first method is added. After that, this call does not return a JSON response body.

[.api]
[field]#recoveryCodes# [type]#[Array]#::
A list of recovery codes. These may be used to disable Multi-Factor authentication at login. If a recovery code is used, all methods are removed. If, after that, a Multi-Factor method is added, a new set of recovery codes will be generated.
