== Create a User

This API is used to create a new FusionAuth User from a SCIM request.

=== Request

[.api-authentication]
link:/docs/v1/tech/apis/authentication#api-key-authentication[icon:lock[type=fas]] Create a User from a SCIM request
[.endpoint]
.URI
--
[method]#POST# [uri]#/api/scim/resources/v2/Users#
--

:http_method: POST
include::docs/v1/tech/apis/scim/_scim-user-request-body.adoc[]

=== Response

The response for this API contains the User that was just created. The password, salt and other sensitive fields will not be returned on the API response.
