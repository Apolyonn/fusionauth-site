== Add a Two Factor Method - SMS


SMS two factor is not enable by default on the `tenant` level. To enable please navigate to `Tenants > icon:edit[type=fas] Edit Tenant > Multi-Factor Tab`

image::account-management/enable-sms-tenant.png[Configure Tenant SMS, width=1200]

Here you will be asked to select:

- The `Enabled` toggle
- A `messenger` (previously created - link:/docs/v1/tech/messengers[documentation])
- A `template` (FusionAuth ships with a default to be customized if needed)

Now when you:

1. Navigate back to your account page.
2. Click `Manage two-factor`
3. Click `icon:plus[] Add two-factor`
4. There will be an option for SMS.

image::account-management/sms-configure-cropped.png[Add SMS Authenticator, width=1200]

Next:

1. Enter your phone number.
2. Click on  `icon:arrow-circle-right[type=fas] Send a one-time-code`.
3. Enter the `Verification Code` click `icon:save[type="fas"] Enable`.

icon:trophy[] Success! +

Upon the next login, the user will be prompted for a code sent by SMS code in addition to your password.

=== See It in Action

With SMS MFA enabled, if you log out and log back in you will be presented with the following screen in addition to the typical login screen.
