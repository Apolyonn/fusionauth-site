Download FusionAuth

Get started in seconds - installs with a single command.
This is the full version of FusionAuth, not a feature-crippled "developer" release.

  • Linux
  • macOS
  • Windows
  • Docker

Linux

Copy and paste the code below and we'll do the rest.

Installs in your current working directory using ZIP packages Copied

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh -s - -z"

Installs for all users on the system using DEB or RPM packages Copied

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh"

macOS

Copy and paste the code below into Terminal and we'll do the rest.

Installs in your current working directory using ZIP packages Copied

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh"

Or install using Homebrew using the following commands.

Install the FusionAuth tap, this will only need to be performed once Copied

brew tap fusionauth/homebrew-fusionauth

Install FusionAuth Copied

brew install fusionauth-app

Start FusionAuth Copied

brew services start fusionauth-app

Windows

Copy and paste the code below into Powershell and we'll do the rest.

Installs in your current working directory using ZIP packages Copied

. { iwr -useb https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.ps1 } | iex; install
