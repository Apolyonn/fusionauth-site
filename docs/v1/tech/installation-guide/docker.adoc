== Running in Docker

All of the FusionAuth Docker images may be found on https://hub.docker.com/u/fusionauth/[Docker Hub].

=== FusionAuth App
```
docker pull fusionauth/fusionauth-app
```

=== FusionAuth Search
```
docker pull fusionauth/fusionauth-search
```
